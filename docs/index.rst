ProLint documentation!
======================

Read, analyze and visualize lipid-protein interactions using ProLint.

ProLint is a lightweight python library that automates and accelerates analysis and
visualization of lipid-protein interaction data. With only a few lines of code you can
gain deep insight into the data and visualize interactions in a varietey of formats.
